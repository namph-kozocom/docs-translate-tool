## 3\. マニュアルの読み方

本マニュアルで使用しているアイコンや用語の意味は、以下のとおりです。

### マニュアルで使用しているアイコン

**アイコン**

**内容**

必ず行わなければならないことや、特に留意すべきことが記載されています。

知っていると便利な情報や補足情報が記載されています。

### マニュアルで使用している用語

**用語**

**内容**

アクワイアラ

国際ブランド加盟店の獲得や管理などを行うカード会社です。

印刷会社

Xardの依頼に応じて、Physicalカードなどの郵送物を印刷して発送する会社です。

インフキュリオン

Xardの運営会社です。  
国際ブランドカードの発行処理、決済などのカード発行業務に関するプロセシングを行います。

国際ブランド

国際ブランドネットワークの運営会社です。

国際ブランド加盟店

Xardが発行したカードが利用可能な国際ブランド加盟店です。

国際ブランドメンバー（BINスポンサー）

国際ブランドカードの発行に必要なBIN（銀行識別番号）またはIIN（発行者識別番号）を提供する、国際ブランドメンバーの会社です。

パートナー

Xardのサービス提供を受ける提携会社です。  
パートナーはユーザーの獲得や会員管理を行います。

ユーザー

パートナーが獲得したカード利用者（法人、個人事業主または個人）です。

リアルタイムスイッチング

パートナーにて残高管理を行う、決済処理の方式です。パートナー独自のFunding Sourceを、決済時にリアルタイムで使用します。
