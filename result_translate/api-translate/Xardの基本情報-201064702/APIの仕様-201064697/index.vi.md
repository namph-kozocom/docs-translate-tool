API được sử dụng để thực hiện các tác vụ Xard khác nhau.Để thực thi API, vui lòng tham khảo tài liệu tham khảo API "API-DOC".

URL, ID đăng nhập và mật khẩu cho "API-DOC" như sau.

**URL**

[https://api-doc.connect.xard.jp/](https://api-doc.connect.xard.jp/)

**ID đăng nhập**

xard

**Mật khẩu**

sự tò mò vô hạn

* Trong Xard, quá trình đăng ký/cập nhật được thực hiện không đồng bộ, do đó có thể mất một chút thời gian để hoàn tất quá trình trong điều kiện tải tạm thời cao.
Ví dụ: tùy thuộc vào thời gian xử lý, chẳng hạn như khi xử lý [Tạo doanh nghiệp] (https://api-doc.connect.xard.jp/#Operation/getBusiness) hoặc [Tạo người dùng] (https://api-doc.connect.xard.jp/#Operation/createUser) sau [Tạo doanh nghiệp] (https://api-doc.connect.xard.jp/#Operation/createBusiness), có thể có trường hợp dữ liệu đích không tồn tại và xảy ra lỗi.

*Vui lòng cân nhắc việc thử lại quá trình xử lý khi thiết kế hệ thống của bạn.
