Vui lòng sử dụng API sau để biết trạng thái Doanh nghiệp, Người dùng, Thẻ và Tài khoản.

* Kinh doanh: [API "Nhận kinh doanh"](https://api-doc.connect.xard.jp/#Operation/getBusiness)

* Người dùng: API ["Nhận người dùng"](https://api-doc.connect.xard.jp/#Operation/getUser) hoặc [API "Nhận danh sách người dùng"](https://api-doc.connect.xard.jp/#Operation/getUserList)

* Thẻ: [API "Nhận thẻ"](https://api-doc.connect.xard.jp/#Operation/getCard) hoặc [API "Nhận danh sách thẻ"](https://api-doc.connect.xard.jp/#Operation/getCardList)

* Tài khoản: [API "Nhận tài khoản"](https://api-doc.connect.xard.jp/#Operation/getAccount)
