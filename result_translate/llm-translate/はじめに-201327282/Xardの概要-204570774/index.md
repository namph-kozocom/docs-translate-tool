**Xard（エクサード）**は自社オリジナルのVisaカードまたはJCBカードを簡単＆低コストで発行できる次世代カード発行プラットフォームです。

多様な発行形態で、Fintech企業、金融機関、小売事業者、WEBサービス事業者など、形態を問わず様々な企業のビジネスニーズに対応します。

![](https://infcurion-jira.atlassian.net/wiki/download/attachments/204570774/Overview-01.png?api=v2)

Xardが提供する様々なAPI機能を自社のサービスに組み込むことで、新しい体験を生み出します。インフキュリオンは決済・金融領域における豊富な知見のもと、お客様とともにビジネスをデザインします。
