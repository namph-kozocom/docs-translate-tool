## 1. Target Users

This manual is intended for partners who implement Xard's environment setup and API connections, explaining the methods of introducing Xard.

* * *

## 2. Manual Structure

This manual consists of the following chapters:

[Introduction](https://infcurion-jira.atlassian.net/wiki/spaces/XardDocs/pages/201327282)

Explains the overview of Xard's services and the basic information of this manual.

[Basic Information of Xard](https://infcurion-jira.atlassian.net/wiki/spaces/XardDocs/pages/201064702/Xard)

Explains the flow from card issuance to reconciliation, the components of Xard, card and API specifications, and other basic information about Xard.

[Preparation for Using APIs](https://infcurion-jira.atlassian.net/wiki/spaces/XardDocs/pages/201064743/API)

Explains the environment used by partners and the preparations necessary to use APIs in each environment.

[Function Explanation](https://infcurion-jira.atlassian.net/wiki/spaces/XardDocs/pages/201064724)

Explains the flow of various operations performed in Xard and the APIs used.

[Frequently Asked Questions](https://infcurion-jira.atlassian.net/wiki/spaces/XardDocs/pages/201588991)

Explains frequently asked questions about Xard and their answers.
